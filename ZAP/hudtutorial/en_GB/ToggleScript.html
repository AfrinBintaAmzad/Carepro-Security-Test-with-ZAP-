Toggle Script

ZAP Scripts are a powerful way to take full advantage of ZAP. They can be used to create custom scans, generate alerts, generate payloads for fuzzing and much more.

A few example scripts from the community can be found here

The Toggle Script tool allows you to select a script and toggle it on and off.

It is not shown by default so you will need to add it to a panel as described in the Tool Configuration tutorial page.

Once added, you can either right- or left-click on the Select Script icon to select a script.

After selecting a script, you will see its current status:

  • Enabled:
  • Disabled:

By left-clicking, you can toggle the script to its opposite state. For example, an Enabled script will become Disabled

By right-clicking, you can either change the script or remove this tool from the panel as described in the Tool Configuration tutorial page.
